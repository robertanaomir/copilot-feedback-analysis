# Copilot Feedback Analysis

Projeto de análise de feedbacks de clientes bancários utilizando IA Generativa para identificar padrões, sentimentos e oportunidades de melhoria.

---

## Objetivo

Analisar comentários de clientes bancários para identificar:

- principais reclamações
- elogios
- sentimentos
- oportunidades de melhoria
- recomendações para áreas de negócio

---

## Tecnologias

- Microsoft Copilot
